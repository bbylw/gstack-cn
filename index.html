// 核心技能

18 位专家，七种强力工具

每个技能都输入到下一个，形成完整的开发冲刺流程。

💭

/office-hours

YC 办公时间 — 六个强制性问题，在你写代码之前重新框定产品

📋

/plan-ceo-review

CEO 视角 — 重新思考问题，找到隐藏的 10 星产品机会

🏗️

/plan-eng-review

工程经理 — 锁定架构、数据流、边界情况和测试计划

👀

/review

高级工程师 — 找到通过 CI 但在生产爆炸的 bug

🧪

/qa

QA 负责人 — 真实浏览器测试，自动修复并生成回归测试

🚀

/ship

发布工程师 — 同步、测试、审计覆盖率、推送、打开 PR

🌐

/browse

QA 工程师 — 给代理眼睛，真实 Chromium 浏览器和截图

🤖

/codex

第二意见 — OpenAI Codex CLI 的独立代码审查

// 快速开始

30 秒安装，5 分钟上手

要求：Claude Code、Git、Bun v1.0+、Node.js（Windows）

terminal — bash
1 # 安装 gstack
2 $ git clone https://github.com/garrytan/gstack.git ~/.claude/skills/gstack
3 $ cd ~/.claude/skills/gstack && ./setup
4
5 # 开始使用
6 $ /office-hours # 描述你正在构建的内容
7 $ /plan-ceo-review # 功能想法审查
8 $ /review # 代码审查
9 $ /qa https://staging.myapp.com # 预发布测试
10 $ /ship # 发布 PR
