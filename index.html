快速开始

30 秒安装，5 分钟上手

要求：Claude Code、Git、Bun v1.0+、Node.js（Windows）

terminal — bash
# 安装 gstack
$ git clone https://github.com/garrytan/gstack.git ~/.claude/skills/gstack
$ cd ~/.claude/skills/gstack && ./setup
# 开始使用
$ /office-hours # 描述你正在构建的内容
$ /plan-ceo-review # 功能想法审查
$ /review # 代码审查
$ /qa https://staging.myapp.com # 预发布测试
$ /ship # 发布 PR
